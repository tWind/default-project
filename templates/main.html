Описание «Default Project»

Данный инструмент является стартовым шаблоном для простых проектов. Он отлично подходить для верстки любых сайтов, которые не являются полноценными веб-приложениями.

Файловая структура:

  • grunt/ Каталог с тасками для Grunt
    • aliases.yaml Файл ассоциаций, в котором зарегистрированы все алиасы задач. Это то, что мы должны сделать перед тем, как вызывать registerTask функцию.
    • autoprefixer.js Таск добавляет только префиксы для требующих поддержки браузеров.
    • browserify.js Позволяет использовать require в браузере также, как это делается в Node. Все указанные зависимости собираются в один файл.
    • concat.js Таск для простого объединения файлов. Все указанные зависимости собираются в один файл.
    • connect.js Простой веб-сервер для статических сайтов
    • copy.js Копирует указанные директории или файлы. Необходим для создания каталога /public/dist/ , в котором храняться обработанные версии файлов для продакшена.
    • imagemin.js Оптимизирует изображения в каталоге /public/dist/ .
    • sass.js Сборка одного CSS файла из всех импортированных SASS/SCSS файлов.
    • sprite.js Сборка спрайтов
    • watch.js Отслеживание изменений. При необходимости запускает те или иные таски.
  • node_modules/ Директория модулей Node
  • public/ В каталоге храняться исходники и скомпилированные версии стилей, скриптов и изображений
    • blocks/ Это логически независимые единицы макета страницы. Блок содержит .scss и .js файлы, описывающие его стили и функциональные возможности. Файлы стилей начинаются с префикса "b-", далее следует название блока, например "b-button".
    • dist/ Хранит в себе обработанные версии файлов для продакшена.
    • fonts/ Шрифты подключенные через @font-face.
    • img/ Изображения. В том числе иконки и спрайты.
    • libs/ Сторонние JS/SCSS библиотеки и небольшой JS-фреймворк для облегчения рутинных операций.
    • default-project.bundle.js Запуск небольшого фреймворка Wind
    • default-project.scss Импорты SASS/SCSS из blocks/ и libs/
  • templates/ Статичные шаблоны в HTML
